# Southern Guardian K9

Southern Guardian K9 is a fictional premium dog gear brand website created as part of an AI Essentials portfolio project.

#Project Overview
This project was designed to demonstrate the use of AI-assisted design, branding, and front-end web development concepts. The website features a military-inspired, premium aesthetic centered around loyalty, protection, and working dog equipment.

#Features
Responsive homepage layout
Custom logo and branding
Hero banner using a real Cane Corso image
Featured product section
Navigation menu
Custom HTML and CSS styling
AI-assisted content and design workflow

#Technologies Used
HTML5
CSS3
GitHub
AI-assisted design and content generation

#Purpose
This project demonstrates foundational website development, branding, and design principles while building a professional technical portfolio.

Note: Southern Guardian K9 is a fictional company created for educational purposes.

- `index.html`
